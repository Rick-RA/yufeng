# JSX

JSX中的变量或者表达式用 大括号"{}"来包裹。

// 放置变量
const name = 'Rick'
const element = <h1>Hello {name}</h1>

// 放置表达式
function add(p1, p2) {
  return p1 + p2
}
const p1 = 3, p2 = 5
const element = <h1>Hello {add(p1, p2)}</h1>

为了便于阅读建议将内容拆分成多行，并放到小括号"()"之间，如下：

const name = 'Rick'
const element = (
	<h1>
  	Hello {name}
  </h1>
)

JSX本身是一个表达式

function getGreeting(user) {
  if (user) {
    return <h1>Hello, {formatName(user)}!</h1>;
  }
  return <h1>Hello, Stranger.</h1>;
}

JSX特定属性

  • 引号 使用引号，将属性值指定为字符串字面量

    const element = <div tabIndex="0"></div>;

  • 大括号

    使用大括号，插入一个表达式

    const element = <img src={user.avatarUrl}></img>;

  • 对于同一属性不能同时使用大括号和引号

  • JSX语法上应更接近Javascript，而不是HTML，例如变量命名应采用小驼峰。

使用JSX指定子元素

如果一个标签里面没有内容，那么可以使用 />来闭合标签

const element = <img src={user.avatarUrl} />;

一个JSX元素可以包含多个子元素。

JSX防止注入攻击

const title = response.potentiallyMaliciousInput;
// 直接使用是安全的：
const element = <h1>{title}</h1>;

React Dom在渲染所有输入内容之前，默认都会进行转义。所有内容在渲染之前都会被转换成了字符串。这样可以有效抵防止XSS攻击。

JSX表示对象

Babel会把JSX转译成一个名为React.CreateElement()函数的调用。

一下两种方式是等效的：

const element = (
  <h1 className="greeting">
    Hello, world!
  </h1>
);
const element = React.createElement(
  'h1',
  {className: 'greeting'},
  'Hello, world!'
);

实际上会创建下面这样一个对象：

const element = {
  type: 'h1',
  props: {
    className: 'greeting',
    children: 'Hello, world'
  }
}
