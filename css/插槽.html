# 插槽

插槽内容：

<ul>
  <li v-for="item in todoList" :key="item">
  	<slot name="todo-item" :todoItem="item"></slot>
  </li>
</ul>

# 新的指令v-slot

<todo-list v-slot:todo-item="slotProps">
	{{slotProps.todoItem}}  
</todo-list>

# 老指令 slot slot-scope

<todo-list slot="todo-item" slotScope="slotProps">
	{{slotProps.todoItem}}  
</todo-list>

# 总结：

新指令主要是对老的指令写法进行的精简，原来的slot表示插槽的名称，slotScope表示插槽传过来的值。

现在由一个指令v-slot来完成。
