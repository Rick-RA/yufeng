# 转换方法

  • toLocaleString()

  • toString()

    • 返回由数组中每个值的字符串形势拼接而成的一个以逗号分隔的字符串。类似于join，但是分隔符是逗号

  • valueOf()

    • 返回数组本身

  • Array.prototype.join()

    • 将一个数组（或一个类数组对象）的所有元素连接成一个字符串并返回这个字符串。如果数组只有一个项目，那么将返回该项目而不使用分隔符。

# 栈方法

  • push() 返回数组长度

# 队列方法

  • shift() 从数组前面删除一个元素，返回被删除的item
  • unshift() 从数组前面添加元素，返回数组长度

# 排序

  • reverse() 用于翻转数组

  • sort()

    • 方法会调用每个数组的toString方法，然后比较得到的字符串。即使每一项都是数值，sort比较的也是字符创
    • 可以接受一个条件函数作为参数，来比较数组
    • 默认是升序排序

# 操作

  • slice()

    • 接收1或2个参数，即要返回下项的起始和结束位置，但不包括结束位置的那一项

  • splice()

    • 第一个参数表示起始位置
    • 第二个参数表示要删除的长度
    • 第三个参数表示要添加的内容（该参数可有可无）

# RegExp

# var expression = / pattern /flags

# flags

  • g

    • 表示全局模式，即模式将被应用于所有的字符串，而非在发现第一个匹配项时立即停止

  • i

    • 表示不区分大小写模式，即在匹配项时忽略模式与字符串的大小写

  • m

    • 表示多行模式，即在到达一行文本末尾时还会继续查找下一行中是否存在模式匹配的项

# 创建正则表达式

  • 以字面量形式创建

    • var pattern = /[bc]at/i

  • 使用RegExp构造函数

    • var pattern = new RegExp("[bc]at","i")

# 实例属性

  • global

    • 布尔值，表示是否设置了g标志

  • ignoreCase

    • 布尔值，表示是否设置了i标志

  • lastIndex

    • 整数，表示开始搜索下一个匹配项的字符位置，从0算起

  • multiline

    • 布尔值，表示是否设置了m标志

  • source

    • 正则表达式的字符串表示，按照字面量形式而非传入构造函数中的字符串模式返回

# 实例方法

  • exec

    • 接收一个参数，即要应用模式的字符串

    • 返回第一个匹配项信息的数组，没有匹配项时返回null

      • index 表示匹配项在字符串中的位置
      • input 表示应用正则表达式的字符串
      • 数组

  • test

    • 接收一个参数，即要应用模式的字符串
    • 在模式与该参数匹配的情况下返回true，否则返回false

# 函数声明与函数表达式

  • 解析器会率先读取函数声明，并使其在执行任何代码之前可用
  • 表达式，必须等解析器执行到它所在的代码行，才会真正被解析执行

# arguments

  • 类数组对象，保存着传入函数中的所有参数

# this

# length

  • 函数希望接收的命名参数的个数

# prototype

  • 保存了所有引用类型的实例方法的真正所在

# 每个函数都有两个非继承方法

  • apply()

    • 接收两个参数

      • 1、函数运行作用域
      • 参数的数组（可以是Array的实例，也可以是arguments对象）

  • call()

    • 接收两个参数

      • 1、函数运行作用域
      • 2、传给函数的参数必须逐个列出来，不能像apply一样可以传递一个Array或者arguments

  • bind()

    • 创建一个函数实例，其this值会被绑定到传给bind()的参数
    • 和apply和call的区别是，它返回的是一个函数，并且默认不会调用。
