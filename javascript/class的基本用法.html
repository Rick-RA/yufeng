# 4、构造函数的prototype在ES6的类上继续存在，类的所有方法都定义在类的prototype上

# 5、Object.assgin()可以方便的一次向类添加多个方法

# 静态方法

# 类的方法前面加static关键字，表示该方法是一个静态方法。（静态方法只能被类调用，不能被类的实例调用，子类可以继承父类的静态方法，也可以在子类中通过super关键字直接调用）

# 静态属性和实例属性

# 静态属性

  • 旧的写法，ES6规定，Class内部只有静态方法，没有静态属性。（静态属性的定义形式:Class.props = 1）
  • 新的写法，直接在实例属性前面加个static关键字就可以了

# 实例属性

  • 实例属性可以直接写在class中
