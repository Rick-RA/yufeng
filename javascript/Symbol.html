# Symbol

# 概述

# 提出背景：为了防止命名冲突。

# ES5的对象属性名都是字符串，合并对象等操作时容易造成命名冲突

# ES6引入数据类型Symbol，表示独一无二的值

# Symbol是一种类型，不是对象，所以不能使用new命令

# 接受一个字符串作为参数，主要用来在控制台进行和其他Symbol区分

# 如果参数是一个对象，会调用该对象的toString（）方法，将其转为字符串

# 相同的symbol的值是不等的

# 不能和其他类型进行运算

# 可以显示转为字符串类型String(symbol)

# 可以转为字符串Boolean(symbol)

# 作为属性名的Symbol

# a[mySymbol] = 'hello'

# a{[mySymblo]:'hello'}

# 作为属性名时，不能使用点运算符

# 可以用于定义一组常量，保证这组常量的值都是不相等的

# 作为属性名时，该属性是公开属性，不是私有属性

# 属性名遍历

# Object.getOwnPropertySymbols方法可以获取指定对象的所有Symbol属性名
