# 箭头函数

# 如果箭头函数的代码块部分多于一条语句，就要使用大括号将其括起来，并使用return语句返回

# 由于大括号被解释为代码块，如果箭头函数直接返回一个对象，必须在对象的外面加上括号。 let getTempItem = id => ({id:id,name:name})

# 注意事项

  • 函数体内的this对象就是定义时所在的对象，而不是使用时所在的对象
  • 不可以当做构造函数，不可以使用new指令
  • 不可以使用arguments对象，该对象在函数体内不存在。如果需要可以用rest参数代替
  • 不可以使用yeild命令，因此箭头函数不能作为Generator函数
  • 箭头函数里面的this绑定定义时函数所在的作用域
  • 箭头函数没有自己的this，因此不能调用cal() apply() bind()这些方法去改变this的指向
