# Object.assgin()

# 用于将源对象的所有可枚举属性复制到目标对象

# 基本用法

  • 第一个参数是目标对象，后面是源对象

  • 如果目标对象与源对象有同名属性，或多个源对象有同名属性，则后面的属性会覆盖前面的属性

  • 1、如果只有一个参数，会直接返回该参数

  • 2、如果参数不是对象，会将参数转为对象，然后返回

  • 3、undefined和null无法转成对象，会报错

  • 4、如果非对象出现在源对象的位置（不是第一个参数）

    • 1、首先转换成对象
    • 2、如果不能转成对象也不会报错，如undefined，null会跳过

  • 5、Object.assgin()的复制的属性是有限制的，只复制源对象的自身属性（不复制继承属性），也不复制不可枚举属性

# 注意点：

  • Object.assgin()是浅拷贝，不是深拷贝
  • 如果遇到同名属性，处理的结果是覆盖而不是添加

# 常见用途

  • 为对象添加属性

  • 为对象添加方法

  • 克隆对象（深拷贝）

    • Object.assgin({},origin)

  • 合并多个对象

  • 为属性指定默认值

# 属性的遍历

# ES6一共5中方法可以遍历对象的属性

  • for..in

    • 遍历自身和继承的可枚举属性

  • Object.keys()

    • 包括对象自身，不包括继承的可枚举属性

  • Object.getOwnPropertyNames()

    • 包括对象自身的所有属性，包括不可枚举属性，不含symbol

  • Object.getOwnPropertySymbols()

    • 包含对象自身的所有symbol属性

  • Reflect.ownKeys()

    • 包含对象自身的所有属性，不管属性名是Symbol还是字符串，也不管是否可枚举

  • 以上5种都遵循以下遍历顺序

    • 1、首先遍历属性名为数值的属性，按照数字排序
    • 2、遍历所有属性名为字符串的属性，按照生成时间排序
    • 3、最后遍历所有属性名为Symbol值的属性，按照生成时间排序
