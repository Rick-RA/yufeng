# MacOS Flutter开发环境配置

# 1. 下载安装包

git clone -b stable git@github.com:flutter/flutter.git

把clone下来的代码放到你喜欢的目录下，比如放在桌面（/Users/xxx/Desktop/）下面

# 2. 配置环境变量

# *2.1 mac上用的终端默认是bash
vim ~/.bash_profile

如果没有.bash_profile文件，则创建一个。在打开的.bash_profile文件中加入以下环境变量：

export PUB_HOSTED_URL=https://pub.flutter-io.cn
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
export PATH=`pwd`/flutter/bin:$PATH

将上面输入的

export PATH=`pwd`/flutter/bin:$PATH

中的`pwd`替换成你在步骤1中flutter的位置（/Users/xxx/Desktop/）保存，退出。

# *2.2 mac上用的终端是zsh
vim ~/.zshrc

如果没有.zshrc，则创建一个。在打开的.zshrc文件中加入以下环境变量：

export PUB_HOSTED_URL=https://pub.flutter-io.cn
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
export PATH=`pwd`/flutter/bin:$PATH

将上面输入的

export PATH=`pwd`/flutter/bin:$PATH

中的`pwd`替换成你在步骤1中flutter的位置（/Users/xxx/Desktop/）保存，退出。

# 3. 执行检查

flutter doctor
