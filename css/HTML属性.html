# HTML属性

# title

属性规定了关于元素的额外信息。这些信息通常会在鼠标移到元素上时显示一段工具提示文本。

# tabindex

规定了tab键的控制顺序。

<a href="http://www.w3school.com.cn/" tabindex="2">W3School</a>
<a href="http://www.google.com/" tabindex="1">Google</a>
<a href="http://www.microsoft.com/" tabindex="3">Microsoft</a>

# style

规定了元素的行内样式，会覆盖属性任何全局样式。

# contenteditable

规定元素的内容是否可编辑

true表示可编辑

false表示不可编辑

<element contenteditable="true|false">

# spellcheck

<p contenteditable="true" spellcheck="true">这是一个段落。</p>

规定是否对元素进行拼写和语法检查

true表示进行检查

false不检查

# lang

固定了元素内容的语言。

<p lang="fr">Ceci est un paragraphe.</p>

究竟html标签中的lang属性是干嘛的呢？为什么要指定它的值是”en”或者”zh-CN”呢？

1.不写lang属性，整个HTML文档的显示也是没有任何问题的

2.我们都知道，如果你在中国，本地浏览器的默认语言是中文，当我们打开一个外国网站时，浏览器会提示”是否将网页内容翻译成中文”，如下图

# id

id属性规定了HTML元素的唯一的id

id在HTML文档中必须唯一

可通过getElementId来获取对应的元素

# hidden

设置为hidden属性的元素将不会被显示

<p hidden>这个段落应该被隐藏。</p>

# draggable

规定元素是否可拖动。

链接和图像默认是可拖动的。

值 描述
true 可拖动
false 不可拖动
auto 使用浏览器默认行为

# data-*

data-* 属性用于存储页面或应用程序的私有自定义数据。

data-* 属性赋予我们在所有 HTML 元素上嵌入自定义 data 属性的能力。

存储的（自定义）数据能够被页面的 JavaScript 中利用，以创建更好的用户体验（不进行 Ajax 调用或服务器端数据库查询）。

data-* 属性包括两部分：

  • 属性名不应该包含任何大写字母，并且在前缀 "data-" 之后必须有至少一个字符
  • 属性值可以是任意字符串

# dir

规定元素内容的文本方向

<p dir="rtl">Write this text right-to-left!</p>
值 描述
ltr 默认，从左向右的文本方向
rtl 从右向左的文本方向
