# 如何提高网页性能

当使用浏览器访问页面时，除了页面的美观和易用性外，网页的访问速度也直接影响着用户的体验。

可以从以下几个方面来对网页的性能进行优化：

  • 在接口设计合理的情况下，尽量减少http接口请求

  • 使用工具压缩js、css、和图片等静态资源文件

  • 代码封装重用，减少重复的文件

  • 需要的地方使用懒加载

  • 合理的使用缓存

  • 使用图片地图来减少http请求
